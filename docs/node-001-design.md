# Observation Node Design – FSC-OBS-001

## Node Identifier

FSC-OBS-001

## Location

Canton, Michigan, United States

Exact coordinates will not be published publicly for privacy reasons, but all observations will include precise timestamping and directional metadata.

---

## Purpose

FSC-OBS-001 serves as the first observational platform for Frontier Sky Collective and will function as a multi-sensor monitoring station for aerial and atmospheric phenomena.

---

## Planned Sensor Systems

### Optical Monitoring

Primary function: detection and recording of visible aerial events.

Planned configuration:

- low-light sky camera
- wide-angle lens
- continuous night observation

Possible targets:

- meteors
- aircraft
- satellites
- atmospheric optical phenomena
- unidentified aerial objects

---

### Radio Frequency Monitoring

Primary function: monitoring radio spectrum activity using Software Defined Radio (SDR).

Potential monitoring bands:

- aviation communications
- satellite downlinks
- general RF spectrum scanning

---

### ADS-B Aircraft Tracking

Aircraft position data will be collected locally using an ADS-B receiver.

This allows:

- identification of nearby aircraft
- elimination of conventional explanations for observed aerial events
- correlation between visual observations and aircraft movement

---

### Environmental Monitoring

Environmental data will be logged to provide context for observations.

Potential sensors:

- temperature
- humidity
- pressure
- wind
- cloud cover (when available)

---

## Data Handling

All sensor systems will record data with:

- UTC timestamps
- sensor identifiers
- standardized metadata formats

Raw data will be preserved whenever possible.

---

## Future Expansion Possibilities

FSC-OBS-001 may later incorporate:

- radio astronomy instrumentation
- automated telescope systems
- multi-camera sky arrays
- additional SDR receivers

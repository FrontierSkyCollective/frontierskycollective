# Frontier Sky Collective – Project Overview

## Summary

Frontier Sky Collective is an independent citizen-science initiative dedicated to the systematic observation and analysis of atmospheric, aerial, and radio-frequency phenomena.

The project focuses on collecting structured observational data using affordable consumer and custom-built instrumentation. All observations are documented using consistent metadata standards and analyzed using reproducible methods.

The goal is to contribute transparent observational datasets that can assist in distinguishing conventional aerial phenomena from unexplained events.

---

## Objectives

1. Establish a local multi-sensor observation node in Canton, Michigan.
2. Collect synchronized optical, radio-frequency, aircraft, and environmental data.
3. Develop automated analysis workflows using open-source tools.
4. Release structured datasets and methodologies for public and collaborative analysis.

---

## Initial Research Focus

Phase 1 focuses on establishing a **Minimum Viable Observation Node (MVON)** capable of detecting and documenting:

- visible aerial objects
- satellites and orbital debris
- aircraft traffic
- meteors
- atmospheric optical phenomena
- radio-frequency activity in monitored bands
